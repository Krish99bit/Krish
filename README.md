## Getting Started

### Prerequisites

- Python 3.x installed on your system
- pip (Python package installer)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Krish99bit/Krish.git
cd Krish
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```
